## Typography

### Font Family

**Primary:** Lato (sans-serif)  
**Fallback:** Roboto

Loaded from Google Fonts with `swap` strategy (shows fallback immediately).

### Font Scale

| Size | Weight | Usage |
|------|--------|-------|
| 32px | 300 (Light) | Large headings |
| 22px | 400 (Regular) | Page titles |
| 18px | 500 (Medium) | Section headings |
| 16px | 400 (Regular) | Body text, buttons |
| 14px | 400 (Regular) | Secondary text, form labels |
| 12px | 400 (Regular) | Helper text, captions |

### Font Weights

- **300** — Light (headlines)
- **400** — Regular (body, default)
- **500** — Medium (labels, emphasis)
- **600** — Semi-bold (buttons, strong emphasis)
- **700** — Bold (rare, strong emphasis)

## Usage Guidelines

### Colors
- **Primary Blue (#77c2f7)** — Use for interactive elements (links, focus states, selected)
- **Primary Orange (#FAA61A)** — Use only for primary action buttons
- **Dark Navy (#1A2332)** — Use for header/navigation only
- **Success Green (#9ACF30)** — Use for positive states (success, approved)
- **Error Red (#ED5958)** — Use for errors, warnings, destructive actions

### Typography
- Always use Lato as primary font
- Maintain 1.5x line-height for body text
- Use font-weight 600+ only for emphasis

### Spacing
- Always snap to 4px grid
- Maintain consistent gaps between sections
- Use `md` (16px) as default component gap

### Accessibility
- Maintain minimum 4.5:1 contrast ratio for text
- Error Red and Orange meet WCAG AA standards
- All colors tested for colorblind accessibility
